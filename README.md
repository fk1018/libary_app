# Libary App

## Description

A basic react app created with vite using typescript.

## Installation

```bash
npm install
```

## Usage

To run the app locally:

```bash
npm run dev
```

## Testing

To run the test locally:

```bash
  npm run test
```

To generate test coverage:

```bash
npm run coverage
```

To run with vitest ui:

```bash
npm run test-ui
```
